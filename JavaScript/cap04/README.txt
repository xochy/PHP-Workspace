-Operador de concatenación +
-Operadores de igualdad == y ===
-Plantillas de cadena

Las plantillas de cadena son una alternativa a la concatenación con +

const valorSrc ='Carita' + día + '.png';

es lo mismo que...

const valorSrc = `Carita${día}.png`;

Se utiliza el carácter backtick(`) en lugar de comillas dobles o simples
